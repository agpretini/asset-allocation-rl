# Data Directory

This directory contains generated data and experiment artifacts.

The project uses reproducible scripts to rebuild these files. In general, files
inside `data/processed/` are outputs, not hand-edited source files.

## Structure

```text
data/
├── processed/
│   ├── dataset/
│   ├── benchmarks/
│   ├── baselines/
│   ├── q_learning/
│   ├── walk_forward/
│   ├── optimization/
│   ├── analysis/
│   └── comparison/
```

## Subdirectories

### `processed/dataset/`

Canonical processed market dataset.

Generated by:

```bash
uv run python scripts/build_dataset.py
```

Expected contents:

- `monthly_dataset.parquet`: monthly asset prices, returns, and macro columns.

### `processed/benchmarks/`

Passive benchmark performance reports.

Generated by:

```bash
uv run python scripts/run_benchmarks.py
```

Expected contents:

- `benchmark_report.csv`: metrics for passive benchmark portfolios.

### `processed/baselines/`

Random and rule-based baseline backtests.

Generated by:

```bash
uv run python scripts/run_backtest.py
```

Expected contents:

- `baseline_report.csv`: metrics for random, equal-weight, momentum,
  volatility-weighted, and passive benchmark strategies.
- `baseline_paths.csv`: portfolio value paths for the same strategies.

### `processed/q_learning/`

First RL-agent training and evaluation artifacts.

Generated by:

```bash
uv run python scripts/train_agent.py
```

Expected contents:

- `q_learning_model.json`: saved tabular Q-learning model.
- `q_learning_training_history.csv`: per-episode training statistics.
- `q_learning_report.csv`: evaluation metrics.
- `q_learning_paths.csv`: action, allocation, return, turnover, and value path.

### `processed/walk_forward/`

Walk-forward robustness evaluation.

Generated by:

```bash
uv run python scripts/run_walk_forward.py
```

Expected contents:

- `walk_forward_report.csv`: metrics by walk-forward split.
- `walk_forward_paths.csv`: portfolio value paths by split.

### `processed/optimization/`

Hyperparameter optimization artifacts.

Generated by:

```bash
uv run python scripts/optimize_hyperparameters.py
```

Expected contents:

- `hyperparameter_results.csv`: ranked Q-learning hyperparameter trials.
- `best_hyperparameters.yaml`: best configuration from the search.

### `processed/analysis/`

Portfolio behavior analysis.

Generated by:

```bash
uv run python scripts/analyze_portfolio.py
```

Expected contents:

- `portfolio_analysis.md`: human-readable behavior report.
- `portfolio_allocations.csv`: allocation time series.
- `allocation_summary.csv`: per-asset allocation summary.
- `regime_summary.csv`: performance and allocations by regime.
- `turnover_summary.csv`: turnover diagnostics.
- `drawdown_summary.csv`: drawdown diagnostics.

### `processed/comparison/`

Model-vs-baseline comparison.

Generated by:

```bash
uv run python scripts/compare_results.py
```

Expected contents:

- `model_vs_baselines.md`: readable model comparison report.
- `model_vs_baselines_report.csv`: combined metrics table.
- `model_vs_baselines_deltas.csv`: model-minus-baseline metric deltas.
